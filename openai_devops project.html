In a Node.js script, you can access runtime arguments passed from the command line using the process.argv array. The process.argv array contains the command-line arguments with the first two elements being the path to the Node.js executable and the path to the script being run. The subsequent elements are the command-line arguments.

Here's a simple example:

javascript
// example.js
console.log('Arguments:', process.argv);

// Extracting command-line arguments
const args = process.argv.slice(2); // The first two elements are not relevant to us

console.log('Runtime Arguments:', args);

When you run this script from the command line:

bash
node example.js arg1 arg2 arg3

The output will be:

arduino
Arguments: [
  'C:\\path\\to\\node.exe',
  'C:\\path\\to\\example.js',
  'arg1',
  'arg2',
  'arg3'
]
Runtime Arguments: [ 'arg1', 'arg2', 'arg3' ]

In this example, the slice(2) method is used to remove the first two elements from process.argv, leaving only the command-line arguments.

You can then use these runtime arguments in your script as needed. For example:

javascript
// example.js
const args = process.argv.slice(2);
